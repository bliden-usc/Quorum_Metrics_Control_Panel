Background: How the Contacts, Staff & Officials Lists are built ▼

The lists you download in sections D–E are built in Quorum before each reporting run. Here's the process for building each list type (Staffers, Contacts, or Officials):

Step 1 — Find everyone who participated in Notes

  • Go to Quorum Search → Staffers / Contacts / Officials
  • Apply filter:
    • Interactions → Participated in notes…
    • Note Start Date: In the last month
    • Click Apply All
  • Save this as List A in Metrics Tracking Issue
  • Clear the filter

Step 2 — Find everyone whose office participated in Notes

  • Apply filter:
    • Interactions → Office participated in notes…
    • Note Start Date: In the last month
    • Click Apply All
  • Save this as List B in Metrics Tracking Issue
  • Clear the filter

Step 3 — Find everyone who opened an Outbox email

  • Apply filter:
    • Outbox Emails → Opened Email…
    • Date: In the last month
    • Click Apply All
  • Save this as List C
  • Clear the filter

Step 4 — Combine into one final list

  • Apply filter: Lists = List A + List B + List C
  • Save as the combined list (e.g., "Last month staffers")

Repeat this process separately for Staffers, Contacts, and Officials.
